1.1 Struktur Dokumen HTML

Materi Video: Pengenalan HTML (10:24)

Pendahuluan

HTML (HyperText Markup Language) adalah kerangka dasar untuk membuat sebuah website. Setiap dokumen web dimulai dengan deklarasi doctype, elemen <html>, <head>, dan <body>.

Contoh Kode (Code Snippet)

Berikut adalah contoh kerangka minimum dari sebuah dokumen HTML5 yang valid:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Webpage</title>
</head>
<body>
    <h1>Hello World!</h1>
    <p>Ini adalah paragraf pertama saya.</p>
</body>
</html>

Latihan Mandiri

Cobalah buat satu file index.html di komputer kalian dan tulis kode di atas. Buka dengan browser untuk melihat hasilnya!
