Week 7: Latihan Bootstrap

Deskripsi

Implementasi framework Bootstrap 5.3.3 untuk merancang halaman web yang responsif, modern, dan interaktif. Latihan ini berfokus pada pemanfaatan sistem grid, utilitas flexbox, dan komponen bawaan Bootstrap.

Fitur & Komponen

  • Halaman Konten Responsif: Memanfaatkan Navbar, Carousel, Cards Grid, List Group, dan utilities Bootstrap untuk menyusun layout artikel berita/portal informasi yang ramah perangkat seluler.
  • Halaman Login & Register: Formulir interaktif dengan fitur tab-switching untuk berpindah halaman masuk/daftar secara dinamis, dilengkapi validasi forms bawaan Bootstrap (`was-validated`), floating labels, dan show/hide password.
  • Kustomisasi Tema: Penyelarasan warna visual agar tetap harmonis dengan identitas portfolio yang dominan maroon dan cream.
