Week 13: Penerapan Sistem CRUD pada Final Project

Deskripsi

Dokumentasi fitur CRUD Final Project MieMe yang menampilkan kemampuan customer dan admin dalam mengelola data melalui antarmuka aplikasi.

CRUD Customer

Login

Customer dapat mengakses akun mereka melalui halaman login untuk kemudian menggunakan fitur CRUD lain secara aman.

Edit Detail Profil

Customer dapat mengubah detail profil mereka untuk memperbarui informasi pribadi dalam aplikasi.

Favorit

Customer dapat menandai menu favorit untuk memudahkan akses dan pengelolaan daftar menu pilihan.

Menambahkan Keranjang

Customer dapat menambahkan menu ke keranjang sebagai bagian dari proses pembuatan pesanan baru.

Upload Bukti Pembayaran

Customer dapat mengunggah bukti transfer dari media pada perangkatnya.

Export Invoice transaksi dan CSV Riwayat Transaksi

Customer dapat mengekspor faktur pesanan ke format CSV sebagai bukti transaksi yang dapat disimpan.

CRUD Admin

Dashboard

Admin dapat melihat ringkasan data dan status transaksi sebagai pusat kontrol aplikasi.

Mengubah Status Pemesanan

Admin dapat memperbarui status pesanan untuk melacak proses pemesanan hingga selesai.

Mengubah Status Pembayaran

Admin dapat mengubah status pembayaran untuk mencatat bukti transaksi dan validasi pembayaran.

Mengedit Menu

Admin dapat mengedit data menu untuk memperbarui informasi produk yang ditawarkan.

Mengedit Kategori Menu

Admin dapat mengubah kategori menu untuk menjaga organisasi dan pengelompokan produk.

Melihat Detail Pengguna

Admin dapat melihat detail pengguna untuk memantau dan mengelola akun yang terdaftar.

Mengubah Role Pengguna

Admin dapat mengubah peran pengguna untuk mengatur hak akses dalam sistem.

Menghapus Ulasan Customer

Admin dapat menghapus ulasan customer yang tidak sesuai untuk menjaga kualitas konten platform.

Export CSV Laporan Transaksi

Admin dapat mengekspor laporan transaksi ke format CSV untuk keperluan analisis dan pelaporan.
